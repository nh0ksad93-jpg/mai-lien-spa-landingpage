# Agent Registry — Danh sách Agent & Phạm vi

> File này là "sổ đăng ký" chính thức. Mọi agent phải biết ai làm gì.

---

## MVP Agents

### marketing-strategist
- **Vai trò:** Chief Marketing Strategist
- **Phạm vi:** Chiến lược marketing, brief chiến dịch, phân bổ ngân sách
- **Không làm:** Viết copy, phân tích data chi tiết, quản lý ads
- **Output cho:** Content Planner, Performance Marketing
- **Quyền docs:** CHỈ ĐỌC

### customer-insight-research
- **Vai trò:** Customer & Market Research Analyst
- **Phạm vi:** Nghiên cứu insight, phân tích đối thủ, xây dựng persona
- **Không làm:** Viết content, xây chiến lược tổng, bịa số liệu
- **Output cho:** Strategist, Content Planner
- **Quyền docs:** CHỈ ĐỌC

### content-planner
- **Vai trò:** Content Strategy & Planning Manager
- **Phạm vi:** Content calendar, phân bổ pillars, content brief
- **Không làm:** Viết nội dung chi tiết, quyết định budget ads
- **Output cho:** Copywriter, Social Media Agent
- **Quyền docs:** CHỈ ĐỌC

### copywriter
- **Vai trò:** Creative Copywriter & Content Creator
- **Phạm vi:** Viết social, ad copy, email, blog, landing page, A/B variants
- **Không làm:** Tự chọn chủ đề, publish, bịa testimonial
- **Output cho:** Social Media Agent, Performance Agent
- **Quyền docs:** CHỈ ĐỌC
