# Planning Skill

## Mục đích
Kỹ năng lập kế hoạch cho bất kỳ task phức tạp nào.

## Khi nào dùng
- Khi task có >3 bước
- Khi cần phối hợp nhiều agent
- Khi output ảnh hưởng đến chiến lược hoặc ngân sách

## Agent ưu tiên dùng
Strategist, Content Planner, Operations Coordinator

## Các bước
1. Xác định mục tiêu cụ thể (SMART)
2. Liệt kê input cần thiết
3. Chia thành sub-tasks
4. Xác định thứ tự và dependencies
5. Ước lượng thời gian
6. Xác định checkpoints human review
7. Tạo mini-plan bullet list

## Constraints
- Plan phải fit trong timeframe yêu cầu
- Không lên plan cho việc ngoài scope agent
- Phải reference docs liên quan
