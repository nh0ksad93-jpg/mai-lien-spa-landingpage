# Campaign Brief Skill

## Mục đích
Tạo brief chiến dịch chuẩn để bàn giao giữa các agent.

## Khi nào dùng
- Khi Strategist cần brief cho Content Planner
- Khi bắt đầu chiến dịch mới

## Agent ưu tiên dùng
Strategist, Operations Coordinator

## Template brief
```yaml
campaign_name: string
objective: string
target_persona: string
key_message: string
channels: []
budget: number
timeline: string
success_metrics: []
constraints: []
creative_direction: string
cta: string
reference_materials: []
```

## Constraints
- Brief phải đủ chi tiết để agent tiếp theo làm việc mà không cần hỏi thêm
- Phải reference persona và brand voice docs
- Phải có KPI đo lường rõ ràng
