# Performance Audit Skill (Mở rộng)

## Mục đích
Đánh giá hiệu quả chiến dịch/content dựa trên data.

## Khi nào dùng
- Cuối tuần/tháng (báo cáo định kỳ)
- Khi chiến dịch underperforming
- Khi cần quyết định scale/stop

## Agent ưu tiên dùng
Analytics Agent, Performance Marketing Agent

## Các bước
1. Thu thập data từ các kênh
2. So sánh actual vs target KPI
3. Xác định top/bottom performers
4. Phân tích nguyên nhân
5. Đề xuất tối ưu cụ thể (với expected impact)
6. Tạo báo cáo theo format chuẩn
