# ERRORS.md — Log Lỗi & Pattern Sai

> Ghi nhận mọi lỗi, pattern sai để agent debug và cải thiện dần.
> Cập nhật mỗi khi phát hiện pattern lỗi lặp lại.

---

## Cách ghi log

```
### [Ngày] — [Agent] — [Loại lỗi]
- **Mô tả:** ...
- **Nguyên nhân:** ...
- **Cách fix:** ...
- **Trạng thái:** đã fix / đang theo dõi / chưa fix
```
