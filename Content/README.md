# Hướng Dẫn Tổ Chức Thư Mục Content

Để thư mục `Content/` không bị lộn xộn sau thời gian dài sử dụng, em khuyến nghị anh tổ chức lưu trữ các bài viết do AI tạo ra theo cấu trúc phân cấp thời gian hoặc chiến dịch.

## Cấu trúc đề xuất:

```text
Content/
├── 2026/
│   ├── Thang-05/
│   │   ├── Chien-Dich-X/
│   │   │   ├── bai-viet-facebook-1.md
│   │   │   ├── email-chao-hang.md
│   │   └── Daily-Posts/
│   │       ├── post-ngay-15.md
│   └── Thang-06/
│       └── ...
└── README.md (file này)
```

**Quy tắc:** Khi nhận được content hoàn chỉnh từ AI, hãy lưu file vào đúng thư mục tháng/chiến dịch tương ứng.
